## [1.0.0] - 2025-10-22

### Added

- Initial public release of **BLinkOS Platform**.
- Implemented Next.js 14 App Router with basic pages for dashboard, login, and marketplace.
- Added mock API routes for agents, tools, and stats.
- Included reusable components: `Navbar`, `AgentCard`, `ToolCard`, and `StatsCard`.
- Integrated authentication using NextAuth.js with email/password and OAuth providers.
- Added Tailwind CSS for styling and configured PostCSS and Autoprefixer.
- Provided example environment configuration via `.env.example`.
